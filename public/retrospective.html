        How to run the retrospective

        Use the board to gather notes, vote on priorities, and create follow-up actions.

        Suggested flow

        1. Add short notes to Start, Stop, and Continue.
        2. Use details when a note needs extra context.
        3. Vote on the notes that matter most to you.
        4. Discuss the highest-voted notes with the team.
        5. Create action items for decisions that need follow-up.

        Columns

        Start
        Ideas, behaviours, or practices the team should begin doing.
        Stop
        Things that slow the team down or should be reduced.
        Continue
        Working habits and strengths the team should keep.

        Tools on this page

        • The timer helps the facilitator timebox writing, voting, and discussion.
        • Retro Health shows whether every column has notes and enough votes.
        • Participants shows who is currently connected to the board.
        • The Actions report lists follow-up items created from retro notes.
